2024年，如何使用 github pages + Hexo + Next 搭建个人博客

参考教学

在本地安装hexo

  • 使用npm安装hexo

前提是先安装过Node.js；它自带包管理工具npm（Node Package Manager）。通过npm，可以方便地安装、管理Hexo及其相关插件、依赖项。

配置主题

  • 进入上面创建的blog文件夹中，将使用的主题文件从github克隆到本地
    git clone https://github.com/EvanNotFound/hexo-theme-redefine.git themes/redefine

  • 配置 hexo 的主题参数（在根目录的_config.yml文件中）

    • 修改theme参数为redefine

  • 启动 Hexo 验证主题是否设置成功

将生成的静态页面部署到github上

  • 在自己的github中创建一个新的仓库，以账号名.github.io为仓库名称

  • 修改hexo配置(_config.yml)文件，并将博客推送到自己的github仓库

    • 将repo参数修改为自己仓库的地址
    • 要先安装deploy-git，才能用命令行部署到github
    • 依次执行以下命令

过一会儿就可以在http://yourname.github.io这个网站看到你的博客了！

最后根据自己的喜好，修改主题的配置文件

  • 文件路径为blog\themes\redefine_config.yml
  • 具体的查看该主题的官方文档
